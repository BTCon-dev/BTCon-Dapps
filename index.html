BTCon is a utility token on the TON blockchain that tracks the approximate value of Bitcoin (BTC) without being backed or bridged. It is designed for simulations, Web3 experiments, and educational tools where a BTC-like value reference is useful. BTCon is not a wrapped asset, nor does it represent real BTC.
